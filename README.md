🚀 Projeto Web Dev - Material de Aula
Bem-vindo ao repositório oficial da disciplina! Este espaço foi criado para centralizar todos os códigos, exemplos e exercícios práticos que desenvolveremos em aula.

📋 Pré-requisitos
Antes de começar, você precisará ter instalado em sua máquina:

Visual Studio Code (VS Code): Baixar aqui

Git: Baixar aqui

🛠️ Como configurar o ambiente (Primeira vez)
Se você nunca usou o Git no seu computador, abra o terminal do VS Code (Ctrl + ') e configure sua identidade:

Bash

git config --global user.name "Seu Nome"
git config --global user.email "seu-email@exemplo.com"
📥 Como baixar este material (Clone)
Para ter uma cópia deste projeto no seu computador e acompanhar as aulas:

Abra uma pasta no seu computador onde deseja salvar o curso.

Clique com o botão direito e selecione "Open with Code" (ou abra o VS Code e vá em File > Open Folder).

No terminal do VS Code, digite o seguinte comando:

Bash

git clone https://github.com/carlosteixeiracruz/projeto-web-dev.git
Entre na pasta criada:

Bash

cd projeto-web-dev
🔄 Como atualizar o material (Pull)
Sempre que o professor adicionar um novo exemplo ou exercício durante a aula, você não precisa baixar tudo de novo. Basta rodar este comando no terminal:

Bash

git pull origin main
Isso trará as novidades do GitHub direto para o seu PC.
